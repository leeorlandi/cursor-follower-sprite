Cursor Follower Sprite

A tiny pixel easter egg you can drop into your own web app.

This repo turns the Kings Quest-style cursor follower idea into a small, configurable browser module with live controls for motion, timing, size, and theme.

Inspired by Rob Scanlon’s original homepage easter egg. GitHub here.
